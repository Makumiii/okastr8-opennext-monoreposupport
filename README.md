# Okastr8 OpenNext Monorepo Test

This repository is an end-to-end fixture for Okastr8 monorepo and OpenNext deployments.

- `apps/task-tracker` deploys to `tasktracker.makumitech.dev`
- `apps/take-notes` deploys to `takenotes.makumitech.dev`

Both apps are Next.js App Router applications using SQLite through `sql.js`.
